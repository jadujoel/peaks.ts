Peaks.js

Peaks.js is a JavaScript library that allows you to display and interact with audio waveforms in the browser.

Demo: ClipNode Segment Loop

This demo uses the ClipNode based ClipNodeAudioDriver for playback, and the experimental PixiCanvasDriver for waveform rendering. Add or drag segments on the waveform, then click Loop next to a segment to play it on repeat. Click Stop loop to return to normal playback.
