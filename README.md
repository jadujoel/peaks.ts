# Peaks.ts

[![npm version](https://img.shields.io/npm/v/@jadujoel/peaks.ts.svg)](https://www.npmjs.com/package/@jadujoel/peaks.ts)
[![npm bundle size](https://img.shields.io/bundlephobia/minzip/@jadujoel/peaks.ts.svg)](https://www.npmjs.com/package/@jadujoel/peaks.ts)

Peaks.ts is a TypeScript UI component for displaying audio waveforms. It is a fork of the original Peaks.js library, which was written in JavaScript. The main goal of this project is to provide a modern, type-safe, and maintainable codebase for developers who want to use Peaks.js in their projects.
